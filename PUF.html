Definition

A physically unbreakable function can be likened to a person's fingerprint because it can be used to uniquely recognize an object, due to its specific properties that can change due to operating conditions

A physically unclonable function is a function built into an object, such as an integrated circuit, which when interrogated with a parameter x generates a response y that depends on both x and its unique properties, which are difficult to clone.

Properties-1

Buildability - means that it is easy to build a random instance of a PUF class

Evaluability - this property includes buildability and in addition requires that any instance of a PUF class can be easily evaluated on a random test

Properties-2

Uniqueness - includes evaluability and in addition requires that the results obtained by evaluating the same variable across different PUF instances be different

Unpredictability - includes evaluability and also requires that no PPT algorithm be able to predict the response of a PUF instance for a given value.

Properties-3

Mathematical non-clonability - includes unpredictability if the opponent has resources and unlimited access to a PUF instance in the learning process.

Physical non-clonability - includes evaluability and in addition requires weight in creating a new PUF instance that looks more like another PUF instance than is described by the uniqueness property.

Types

Non-electronic PUFs, their behavior is based on non-electronic technologies or materials.

Electronic PUFs are those constructions that contain random variations in electronic characteristics, for example resistant
Silicon PUFs a major subclass of electronic PUFs that are embedded in a silicon chip.

SRAM PUF

is based on the behavior of standard SRAM memory that is available in any digital chip
Each SRAM cell has its own preferred state each time SRAM is powered, resulting from random differences in threshold voltages.

Therefore, an SRAM response produces a unique and random pattern of 0 and 1. This pattern is like the imprint of a chip because it is unique to a particular SRAM and therefore to a specific chip..

Butterfly PUF

is based on the idea of ​​creating structures within the FPGA matrix that behave similarly to an SRAM cell during the start-up phase

A Butterfly PUF cell is a cross-linked flip-flop, which can be brought into an unstable state before reaching one of two possible stable states. The structure consists of two latches whose outlets are cross-coupled.

Arbitral PUF

is a type of silicon PUF based on delay. The idea behind an arbitrary PUF is to explicitly introduce a race state between two digital paths in a silicon chip.

Both paths end in an arbitrary circuit that has the ability to solve the race, determine which of the two was faster and return the corresponding binary value. Lee proposed implementing the two paths as chains of switch blocks.

Ring PUF

similar to the Arbitral PUF exploits differences in electronic circuits caused by variations in the manufacturing process

PUFs are made up of identical oscillating rings, which are loops of an odd number of inverters, which once stimulated oscillate at a certain frequency. The oscillation frequency can be measured using standard digital components.
